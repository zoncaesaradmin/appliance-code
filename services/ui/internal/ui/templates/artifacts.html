Push from a workstation

Create an appliance API token, then use it as the registry password. Interactive appliance passwords are never accepted by the registry.

podman login <appliance-host> --username <username> --password-stdin
podman push <local-image> <appliance-host>/users/<username>/<repository>:<tag>

Pipe the API token to podman login on standard input. Do not place tokens in command arguments or shell history.
